# MVVMHelper
---
MVVM help based on Android ACC

# 依赖
---
```groovy
dependencies {
    implementation 'com.brave.mvvm.helper:mvvmhelper:0.0.7'
}
```

# 初始化
---
 - 继承[BaseApp](/mvvmhelper/src/main/java/com/brave/mvvm/mvvmhelper/base/CommonApp.kt)
 - 自定义Application添加[BaseApp](/mvvmhelper/src/main/java/com/brave/mvvm/mvvmhelper/base/CommonApp.kt)中的方法到自定义Application中
